# Proposal Tugas Akhir: Food Freshness Detection System

Pengembangan sistem deteksi kesegaran makanan berbasis sensor gas MQ dan machine learning untuk Smart Canteen.

## Deskripsi Singkat
Sistem pemantauan kesegaran makanan secara real-time menggunakan multi-gas sensor untuk mengklasifikasikan makanan sebagai *fresh* atau *spoiled*, mendukung food waste management dan keamanan pangan di kantin.

## Kompilasi
Kompilasi dokumen utama `ProposalTA.tex` menggunakan LaTeX compiler (pdflatex/xelatex).
